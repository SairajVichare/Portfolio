## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Building for Production

```bash
npm run build
```

## Customization

### Personal Information
- Update personal details in `src/components/Hero.js`
- Replace the profile image in `public/profile.jpeg`
- Modify contact information in `src/components/Contact.js`

### Skills
- Edit the skills array in `src/components/Skills.js`
- Add/remove skills and adjust proficiency levels

### Projects
- Update project information in `src/components/Projects.js`
- Add your own project images and links

### Styling
- Main styles are in `src/styles/App.css`
- Customize colors, fonts, and animations as needed
